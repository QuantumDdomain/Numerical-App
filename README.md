# 📊 Numerical Methods Web App

This is a web-based tool built to demonstrate and interact with various **numerical methods** used in computational physics and applied mathematics.

---

## 🛠️ Technologies Used

- Python 🐍
- NumPy
- Matplotlib
- sympy
